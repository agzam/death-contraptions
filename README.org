MCP servers and skills for AI coding assistants.

Aimed to work with various AI fronts - Claude Code CLI, ECA, Claude Desktop, gptel-agent, etc.

* Prerequisites

- [[https://github.com/babashka/babashka][Babashka]] - all .bb servers require it
- [[https://clojure.org/guides/install][Clojure]] - for org-roam-mcp
- Platform tools as needed per the MCP, e.g., kubectl/helm (k8s), osascript (jxa-browser)

* Setup

1. Clone this repo
2. Copy =local-config.example.edn= to =local-config.edn= (or =local-config.edn.gpg= for encrypted)
3. Fill in your personal values (Slack workspace, Splunk host, etc.)
4. Run the setup script:

#+begin_src sh
bb setup.bb
#+end_src

This generates config files and symlinks for all supported consumers:
- =~/.config/eca/config.json= (Claude Code CLI / ECA)
- =~/Library/Application Support/Claude/claude_desktop_config.json= (Claude Desktop, macOS)
- Symlinks =~/.config/eca/tools= and =~/.config/eca/skills= to this repo

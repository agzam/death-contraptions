#+title: splunk

Search Splunk logs, list indexes, and explore metadata. Borrows session credentials from Brave browser cookies on macOS.

macOS only.

* Tools

| Tool                   | Description                                        |
|------------------------+----------------------------------------------------|
| splunk-search          | Run an SPL query with time range and result limits |
| splunk-indexes         | List available indexes with event counts and sizes |
| splunk-search-metadata | Discover hosts, sources, sourcetypes for an index  |

* Prerequisites

- macOS (uses Keychain for cookie decryption)
- Brave browser with an active Splunk session

* Configuration

Settings in =config.edn= (generated by =setup.bb= from =local-config.edn=):

| Key   | Description                                           |
|-------+-------------------------------------------------------|
| :host | Splunk instance hostname (e.g. myorg.splunkcloud.com) |
